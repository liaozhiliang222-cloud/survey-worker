# Cloudflare Pages 自定义域名配置指南

> 目标：将 `survey-worker-711.pages.dev` 替换为你的自定义域名（如 `survey.yourdomain.com` 或 `yourdomain.com`），避免微信拦截。

---

## 前置准备

1. **拥有一个域名**（没有可去阿里云/腾讯云/Namecheap 等注册，推荐 `.com` / `.net`，便宜且通用）
2. **域名 DNS 已托管到 Cloudflare**（必须，下面有迁移步骤）

---

## 第一步：将域名 DNS 托管到 Cloudflare（如已托管可跳过）

如果你域名当前在阿里云/腾讯云/GoDaddy 管理，需要先迁移到 Cloudflare：

1. 登录 [Cloudflare Dashboard](https://dash.cloudflare.com)
2. 点击 **"添加站点"** → 输入你的域名 → 选择 **Free 套餐**
3. Cloudflare 会扫描你现有的 DNS 记录，点击 **继续**
4. 记录页面给出的 **Cloudflare 名称服务器**（通常是 `xxx.ns.cloudflare.com` 两个地址）
5. 去你域名注册商（阿里云/腾讯云等）后台 → 找到 **修改 DNS 服务器** → 替换为 Cloudflare 提供的两个地址
6. 等待 5 分钟~24 小时生效（通常几分钟），Cloudflare 会邮件通知

---

## 第二步：在 Pages 项目绑定自定义域名

1. 打开 [Cloudflare Pages](https://dash.cloudflare.com) → 选择你的 `survey-worker` 项目
2. 点击左侧 **"自定义域"**（Custom Domains）
3. 点击 **"设置自定义域"** → 输入你想用的域名：
   - 推荐用 **子域名**：`survey.yourdomain.com`（容易记，也不影响主域名做其他事）
   - 或直接用 **根域名**：`yourdomain.com`
4. 点击 **"继续"** → Cloudflare 自动检查 DNS 并给出配置建议

### 情况 A：域名已在 Cloudflare 托管
- 直接点击 **"激活域"** → Cloudflare 自动添加 DNS 记录
- 等待 1-2 分钟，显示 **"有效"**（Active）即可

### 情况 B：域名不在 Cloudflare 托管（不建议）
- 需要手动在域名后台添加一条 **CNAME 记录**：
  - 主机记录：`survey`（如果用根域名则填 `@`）
  - 记录类型：`CNAME`
  - 记录值：`survey-worker-711.pages.dev`
- 保存后回 Cloudflare 点击验证

---

## 第三步：验证部署

1. 浏览器访问你的自定义域名（如 `https://survey.yourdomain.com`）
2. 确认能正常打开项目页面
3. 确认 HTTPS 证书已自动配置（Cloudflare 默认自动签发，无需手动操作）

---

## 第四步：微信测试

1. 把新域名 `https://survey.yourdomain.com` 发到微信聊天
2. 点击访问 → 正常应该可以直接打开，不会再被拦截
3. 如果仍提示风险，是因为域名太新/无流量，等 1-2 天自然恢复，或再次申诉

---

## 推荐配置（可选）

### 1. 强制 HTTPS 跳转
在 Cloudflare 域名后台 → **SSL/TLS** → **概述** → 确保模式为 **"完全"** 或 **"完全（严格）"**

### 2. 缓存规则（加速国内访问）
在 Cloudflare 域名后台 → **规则** → **页面规则** → 创建一条：
- URL：`survey.yourdomain.com/*`
- 设置：**缓存级别** → **缓存所有内容**
- 这样静态资源（HTML/CSS/JS）会被 Cloudflare 全球节点缓存，访问更快

### 3. 开启中国网络优化（可选，免费版有限）
Cloudflare 免费版不支持大陆节点加速，但可以通过以下方式提升国内速度：
- 绑定国内域名备案 + 接入 Cloudflare 中国合作伙伴（如百度云加速）—— 较复杂，初期不建议
- 更简单的方案：直接用国内 CDN 如 **又拍云/七牛云** 做一层反向代理（进阶操作）

---

## 常见问题

| 问题 | 解决 |
|------|------|
| 域名已添加但显示 "无效" | 检查 DNS 是否已生效，用 `dig survey.yourdomain.com` 或在线工具查看 CNAME 是否指向 pages.dev |
| 提示 "SSL 证书错误" | 等 5-10 分钟，Cloudflare 自动签发证书；或检查 SSL/TLS 模式是否设为 "完全" |
| 微信还是拦截 | 自定义域名一般 1-3 天内自动解除风控；可再次提交申诉，理由是 "已更换独立域名" |
| 想换另一个域名 | 在 Pages 项目 **自定义域** 里删除旧的，添加新的即可 |

---

## 一句话总结

> 把域名 DNS 切到 Cloudflare → Pages 项目里添加自定义域 → 自动验证通过 → 用新域名访问，微信不再拦截。

**整个流程预计耗时：10-30 分钟（DNS 迁移可能需要几小时生效）。**
